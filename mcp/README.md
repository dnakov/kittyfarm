# KittyFarm MCP

This package exposes KittyFarm to MCP clients over stdio. The MCP process is a sidecar: start the KittyFarm macOS app first, then launch this package from an MCP client.

KittyFarm writes its local API connection file to:

```sh
~/Library/Application Support/KittyFarm/control-api.json
```

The file contains a localhost base URL and bearer token. It is regenerated by the app and should not be committed.

## Development

```sh
npm install
npm run build
```

MCP command:

```sh
node /Users/mweinbach/Projects/kittyfarm/mcp/dist/index.js
```

## Tool Surface

- Device visibility: `kittyfarm_status`, `kittyfarm_list_devices`, `kittyfarm_screenshot`, `kittyfarm_accessibility_tree`
- Device lifecycle: `kittyfarm_connect_device`, `kittyfarm_disconnect_device`
- Input: `kittyfarm_tap`, `kittyfarm_swipe`, `kittyfarm_type`, `kittyfarm_press_home`, `kittyfarm_rotate`, `kittyfarm_open_app`
- Testing: `kittyfarm_find_element`, `kittyfarm_assert_visible`, `kittyfarm_assert_not_visible`, `kittyfarm_wait_for`
- Build/run: `kittyfarm_discover_project`, `kittyfarm_build_and_run`, `kittyfarm_get_logs`
